Command-Line Parameters

-c or --closebutton

Description:Define a place for a close button in virtual console. The button can be used to close QLC. Has effect only in kiosk mode (see -k) and is most useful when in fullscreen mode without a window manager.

Parameters: x,y,w,h

Examples:
Create a button at (x400, y500) whose size is (w70, h50):
qlc -c 400,500,70,50
qlc --closebutton 400,500,70,50

-d or --debug

Description: Enable debug mode and optionally set the output level.

Parameters: Level (see QtMsgType)

Examples:
Enable debug mode and set debug level to 0:
qlc -d
qlc --debug

Enable debug mode and set debug level to 3:
qlc -d 3

-f or --fullscreen

Description: Start the application in fullscreen mode

Parameters: Method (either 'normal' or 'resize')

Examples:
Tell the window manager to give the whole screen space to QLC:
qlc -f
qlc --fullscreen
qlc -f normal
qlc --fullscreen normal

Resize QLC to take up the whole screen space (useful in custom X11 sessions without a window manager):
qlc -f resize
qlc --fullscreen resize

-h or --help

Description: Display command-line help (only in Linux & OS X)

Parameters: None

Examples:
Display the command-line help:
qlc -h
qlc --help

-k or --kiosk

Description: Enable kiosk-mode (only virtual console is visible and the application is locked in operate mode)

Parameters: None

Examples:
Start the application in kiosk mode:
qlc -k
qlc --kiosk

-l or --locale

Description: Use the given locale for translation

Parameters: Locale name (currently supported: en_GB, fi_FI, fr_FR, es_ES)

Examples:
Use finnish language:
qlc -l fi_FI
qlc --locale fi_FI

-o or --open

Description: Open the given workspace file

Parameters: File name

Examples:
Open mydesk.qxw:
qlc -o mydesk.qxw
qlc --open mydesk.qxw

-p or --operate

Description: Start the application in Operate mode.

Parameters: None

Examples:
Start QLC in operate mode:
qlc -p
qlc --operate

-v or --version

Description: Display the current application version number

Parameters: None

Examples:
qlc -v
qlc --version
